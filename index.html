01. Pengertian Enkapsulasi

Enkapsulasi adalah konsep dalam Object Oriented Programming (OOP) yang digunakan untuk menyembunyikan data dan hanya memberikan akses melalui method tertentu.

Constructors Dasar

Constructor dasar merupakan fungsi khusus di dalam sebuah class yang berfungsi untuk memberikan nilai awal pada atribut ketika sebuah objek dibuat. Constructor memiliki nama yang sama dengan class dan akan dipanggil secara otomatis saat objek dibuat, sehingga data pada objek dapat langsung diinisialisasi sejak awal. Berikut contoh penerapan class buku dengan constructors dasar pada dartpad

DART SOURCE CODE

// Nama  : Marsya Dwi Lintang
// NIM   : 251410028
// Kelas : SI2A

class Buku {
  String judul;

  // constructor
  Buku(this.judul);

}

void main() {
  var buku1 = Buku("Pemrograman Berorientasi Objek");
  print(buku1.judul);

}

02. Tujuan Enkapsulasi

  • Melindungi data
  • Mengontrol akses terhadap data
  • Membuat program lebih aman
  • Di Dart dilakukan dengan: Private Field (_), Getter, dan Setter

1. Constructor Mobil

Constructor adalah method khusus pada class yang digunakan untuk membuat objek dan menginisialisasi nilai awal atribut.

DART SOURCE CODE

// Nama  : Marsya Dwi Lintang
// NIM   : 251410028
// Kelas : SI2A


import 'dart:convert';

class Mobil {
  String? merk;
  int? tahun;

  // Parameterized constructor
  Mobil(this.merk, this.tahun);

  // Named constructor
  Mobil.baru() {
    merk = 'Toyota';
    tahun = 2023;
  }

  // Named constructor dengan parameter
  Mobil.dariJson(Map json) {
    merk = json['merk'];
    tahun = json['tahun'];
  }

  @override
  String toString() {
    return 'Mobil{merk: $merk, tahun: $tahun}';
  }
}

void main() {
  print('--- Demonstrasi Class Mobil ---');

  var mobilA = Mobil('Honda', 2020);
  print('Mobil A: $mobilA');

  var mobilB = Mobil.baru();
  print('Mobil B: $mobilB');

  String jsonString = '{"merk": "Suzuki", "tahun": 2021}';
  Map jsonData = jsonDecode(jsonString);
  var mobilC = Mobil.dariJson(jsonData);
  print('Mobil C: $mobilC');

  Map anotherJsonData = {'merk': 'Mitsubishi', 'tahun': 2022};
  var mobilD = Mobil.dariJson(anotherJsonData);
  print('Mobil D: $mobilD');
}

2. Constant Constructor

Constant constructor digunakan untuk membuat objek konstan sehingga nilai objek tidak dapat diubah setelah dibuat.

  String toString() {

  }
}

Praktik 1:Multiple Constructor

Program ini membuat class Mahasiswa yang memiliki beberapa constructor, yaitu constructor default dan named constructor (dariMap() dan tanpaNama()). Tujuannya agar objek mahasiswa bisa dibuat dengan cara yang berbeda. Pada program dibuat 3 objek mahasiswa menggunakan masing-masing constructor.

DART SOURCE CODE

//Nama : Marsya Dwi Lintang
//NIM  : 251410028
//Kelas: SI2A

class Mahasiswa {
  String nama;
  int umur;

  // Constructor default

  Mahasiswa(this.nama, this.umur);

  Mahasiswa.dariMap(Map data)
      : nama = data['nama'],
        umur = data['umur'];

  Mahasiswa.tanpaNama(this.umur) : nama = "Tanpa Nama";

  void tampilkan() {
    print("Nama: $nama");
    print("Umur: $umur");
    print("----------------");
  }
}

void main() {

  var mhs1 = Mahasiswa("Marsya", 20);
 
  var mhs2 = Mahasiswa.dariMap({
    "nama": "Lintang",
    "umur": 21
  });

  var mhs3 = Mahasiswa.tanpaNama(19);
  mhs1.tampilkan();
  mhs2.tampilkan();
  mhs3.tampilkan();
}

7. Getter & Setter dengan Validasi

Getter dan setter dapat digunakan untuk menambahkan validasi agar data yang dimasukkan sesuai aturan.

DART SOURCE CODE

class Produk {
  String _nama;
  double _harga;

  Produk(this._nama, this._harga);

  String get nama => _nama;
  double get harga => _harga;

  set nama(String value) {
    if (value.length >= 3) {
      _nama = value;
    } else {
      print('Nama minimal 3 karakter');
    }
  }

  set harga(double value) {
    if (value > 0) {
      _harga = value;
    } else {
      print('Harga harus lebih besar dari 0');
    }
  }
}

void main() {
  var produk1 = Produk('Monitor Samsung', 5500.0);

  print(produk1.nama);
  print(produk1.harga);

  produk1.nama = 'Keyboard Mekanik';
  produk1.harga = 6200.0;

  print(produk1.nama);
  print(produk1.harga);

  produk1.nama = 'TV';
  produk1.harga = -100;
}

Praktik 2:Enkapsulasi Lengkap

Program ini membuat class User dengan field private (_username, _password, _email). Data diakses menggunakan getter dan setter. Setter memiliki validasi seperti username minimal 5 karakter, password minimal 8 karakter, dan email harus mengandung @, sehingga data lebih aman dan terkontrol.

DART SOURCE CODE

// Nama  : Marsya Dwi Lintang
// NIM   : 251410028
// Kelas : SI2A

class User {
  String _username;
  String _password;
  String _email;
  
  User(this._username, this._password, this._email);

  String get username => _username;
  String get password => _password;
  String get email => _email;

  set username(String value) {
    if (value.length >= 5) {
      _username = value;
    } else {
      print("Username minimal 5 karakter");
    }

  }

  set password(String value) {
    if (value.length >= 8) {
      _password = value;
    } else {
      print("Password minimal 8 karakter");
    }

  }

  set email(String value) {
    if (value.contains("@")) {
      _email = value;
    } else {
      print("Email harus mengandung '@'");

    }

  }

}

void main() {
  var user1 = User("marsya123", "password123", "marsya@email.com");
  print("Username: ${user1.username}");
  print("Password: ${user1.password}");
  print("Email: ${user1.email}");
  print("\n--- Update Data ---");

  user1.username = "lintanggg";
  user1.password = "12345678";
  user1.email = "lintang@gmail.com";
  
  print("Username baru: ${user1.username}");
  print("Password baru: ${user1.password}");
  print("Email baru: ${user1.email}");

}

8. Encapsulation (Login System)

Encapsulation adalah konsep OOP yang membungkus data dan method dalam satu class serta membatasi akses langsung ke data tersebut.

DART SOURCE CODE

// Nama  : Marsya Dwi Lintang
// NIM   : 251410028
// Kelas : SI2A

class User {
  String _username;
  String _password;

  User(this._username, this._password);

  String get username => _username;

  bool login(String username, String password) {
    return _username == username && _password == password;
  }

  void gantiPassword(String lama, String baru) {
    if (_password == lama) {
      if (baru.length >= 8) {
        _password = baru;
        print('Password berhasil diubah');
      } else {
        print('Password minimal 8 karakter');
      }
    } else {
      print('Password lama salah');
    }
  }
}

void main() {
  final user1 = User('admin_user', 'password0443');

  if (user1.login('admin_user', 'password0443')) {
    print('Login berhasil');
  }

  user1.gantiPassword('password0443', 'newPassword456');
}
