01. Pengertian Enkapsulasi

Enkapsulasi adalah konsep dalam Object Oriented Programming (OOP) yang digunakan untuk menyembunyikan data dan hanya memberikan akses melalui method tertentu.

02. Tujuan Enkapsulasi

  • Melindungi data
  • Mengontrol akses terhadap data
  • Membuat program lebih aman
  • Di Dart dilakukan dengan: Private Field (_), Getter, dan Setter

1. Constructor Mobil

Constructor adalah method khusus pada class yang digunakan untuk membuat objek dan menginisialisasi nilai awal atribut.

2. Const Constructor
