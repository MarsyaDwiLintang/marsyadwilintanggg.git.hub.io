01. Pengertian Enkapsulasi

Enkapsulasi adalah konsep dalam Object Oriented Programming (OOP)yang digunakan untuk menyembunyikan data dan hanya memberikan akses melalui method tertentu.

02. Tujuan Enkapsulasi

* Melindungi data * Mengontrol akses data * Membuat program lebih aman Di Dart, enkapsulasi dilakukan dengan: * Private field (_) * Getter * Setter

1. Constructor Mobil

Constructor adalah method khusus pada class yang digunakan untuk membuat objek dan menginisialisasi nilai awal atribut. Dalam Dart terdapat constructor biasa dan named constructor, yaitu constructor dengan nama tertenu yang digunakan untuk membuat objek dengan cara atau kondisi yang berbeda.

DART SOURCE CODE
// Nama  : Marsya Dwi Lintang
// NIM   : 251410028
// Kelas : SI2A

import 'dart:convert';

class Mobil {
  String? merk;
  int? tahun;

  // Parameterized constructor
  Mobil(this.merk, this.tahun);

  // Named constructor
  Mobil.baru() {
    merk = 'Toyota';
    tahun = 2023;
  }

  // Named constructor dengan parameter
  Mobil.dariJson(Map json) {
    merk = json['merk'];
    tahun = json['tahun'];
  }

  @override
  String toString() {
    return 'Mobil{merk: $merk, tahun: $tahun}';
  }
}

void main() {
  print('--- Demonstrasi Class Mobil ---');

  var mobilA = Mobil('Honda', 2020);
  print('Mobil A: $mobilA');

  var mobilB = Mobil.baru();
  print('Mobil B: $mobilB');

  String jsonString = '{"merk": "Suzuki", "tahun": 2021}';
  Map jsonData = jsonDecode(jsonString);
  var mobilC = Mobil.dariJson(jsonData);
  print('Mobil C: $mobilC');

  Map anotherJsonData = {'merk': 'Mitsubishi', 'tahun': 2022};
  var mobilD = Mobil.dariJson(anotherJsonData);
  print('Mobil D: $mobilD');
}
