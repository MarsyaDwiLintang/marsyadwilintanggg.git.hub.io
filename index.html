Pengertian Enkapsulasi

Enkapsulasi adalah konsep dalam Object Oriented Programming (OOP) yang digunakan untuk menyembunyikan data dalam sebuah class dan hanya memberikan akses melalui method tertentu seperti getter dan setter.

Contoh Program Enkapsulasi Produk

Program berikut merupakan contoh penerapan konsep enkapsulasi dalam bahasa pemrograman Dart. Enkapsulasi adalah proses menyembunyikan data di dalam sebuah class dan hanya mengizinkan akses melalui method tertentu seperti getter dan setter.

Pada contoh ini dibuat class Produk yang memiliki atribut nama dan harga. Atribut tersebut dibuat private sehingga tidak dapat diakses langsung dari luar class. Untuk mengakses dan mengubah nilainya digunakan method getter dan setter.

Contoh Program Enkapsulasi User

class User {
String _username;
String _password;

User(this._username, this._password);

String get username => _username;

bool login(String username, String password) {
return _username == username && _password == password;
}

void gantiPassword(String lama, String baru) {
if (_password == lama) {
if (baru.length >= 8) {
_password = baru;
print('Password berhasil diubah.');
} else {
print('Password baru minimal 8 karakter.');
}
} else {
print('Password lama salah.');
}
}
}
