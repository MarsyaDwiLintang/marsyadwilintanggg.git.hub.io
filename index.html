01. Pengertian Enkapsulasi

Enkapsulasi adalah konsep dalam Object Oriented Programming (OOP) yang digunakan untuk menyembunyikan data dan hanya memberikan akses melalui method tertentu.

02. Tujuan Enkapsulasi

  • Melindungi data
  • Mengontrol akses terhadap data
  • Membuat program lebih aman
  • Di Dart dilakukan dengan: Private Field (_), Getter, dan Setter

1. Constructor Mobil

Constructor adalah method khusus pada class yang digunakan untuk membuat objek dan menginisialisasi nilai awal atribut.

2. Const Constructor

Const constructor digunakan untuk membuat objek konstan sehingga nilai objek tidak dapat diubah setelah dibuat.

3. Initializer List

Initializer list digunakan untuk menginisialisasi variabel sebelum constructor dijalankan.

Praktik 1:Multiple Constructor

Program ini membuat class Mahasiswa yang memiliki beberapa constructor, yaitu constructor default dan named constructor (dariMap() dan tanpaNama()). Tujuannya agar objek mahasiswa bisa dibuat dengan cara yang berbeda. Pada program dibuat 3 objek mahasiswa menggunakan masing-masing constructor.

5. Getter

Getter adalah method yang digunakan untuk mengambil nilai dari variabel private.

6. Setter

Setter adalah method yang digunakan untuk mengubah nilai dari variabel private.

7. Getter & Setter dengan Validasi

Getter dan setter dapat digunakan untuk menambahkan validasi agar data yang dimasukkan sesuai aturan.

Paktik 2:Enkapsulasi Lengkap

Program ini membuat class User dengan field private (_username, _password, _email). Data diakses menggunakan getter dan setter. Setter memiliki validasi seperti username minimal 5 karakter, password minimal 8 karakter, dan email harus mengandung @, sehingga data lebih aman dan terkontrol.
