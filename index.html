01. Pengertian Enkapsulasi

Enkapsulasi adalah konsep dalam Object Oriented Programming (OOP) yang digunakan untuk menyembunyikan data dan hanya memberikan akses melalui method tertentu.

02. Tujuan Enkapsulasi

  • Melindungi data
  • Mengontrol akses terhadap data
  • Membuat program lebih aman
  • Di Dart dilakukan dengan: Private Field (_), Getter, dan Setter

1. Constructor Mobil

Constructor adalah method khusus pada class yang digunakan untuk membuat objek dan menginisialisasi nilai awal atribut.

DART SOURCE CODE
// Nama  : Marsya Dwi Lintang
// NIM   : 251410028
// Kelas : SI2A

import 'dart:convert';

class Mobil {
String? merk;
int? tahun;

Mobil(this.merk, this.tahun);

Mobil.baru() {
merk = 'Toyota';
tahun = 2023;
}

Mobil.dariJson(Map json) {
merk = json['merk'];
tahun = json['tahun'];
}

@override
String toString() {
return 'Mobil{merk: $merk, tahun: $tahun}';
}
}

void main() {
print('--- Demonstrasi Class Mobil ---');

var mobilA = Mobil('Honda', 2020);
print('Mobil A: $mobilA');

var mobilB = Mobil.baru();
print('Mobil B: $mobilB');

String jsonString = '{"merk": "Suzuki", "tahun": 2021}';
Map jsonData = jsonDecode(jsonString);
var mobilC = Mobil.dariJson(jsonData);
print('Mobil C: $mobilC');

Map anotherJsonData = {'merk': 'Mitsubishi', 'tahun': 2022};
var mobilD = Mobil.dariJson(anotherJsonData);
print('Mobil D: $mobilD');
}

2. Const Constructor

Const constructor digunakan untuk membuat objek konstan sehingga nilai objek tidak dapat diubah setelah dibuat.

3. Initializer List

Initializer list digunakan untuk menginisialisasi variabel sebelum constructor dijalankan.

4. Private Variable

Private variable adalah variabel yang hanya dapat diakses di dalam class dengan menggunakan tanda underscore (_).

5. Getter

Getter adalah method yang digunakan untuk mengambil nilai dari variabel private.

6. Setter

Setter adalah method yang digunakan untuk mengubah nilai dari variabel private.

7. Getter & Setter dengan Validasi

Getter dan setter dapat digunakan untuk menambahkan validasi agar data yang dimasukkan sesuai aturan.

8. Encapsulation (Login System)

Encapsulation adalah konsep OOP yang membungkus data dan method dalam satu class serta membatasi akses langsung ke data tersebut.
