Pengertian Enkapsulasi

Enkapsulasi adalah konsep dalam Object Oriented Programming (OOP) yang digunakan untuk menyembunyikan data dalam sebuah class dan hanya memberikan akses melalui method tertentu seperti getter dan setter.

Contoh Program Enkapsulasi Produk

print('Harga Produk: Rp ${produk1.harga.toStringAsFixed(2)}');

// Menggunakan setter untuk memperbarui nilai
print('\n--- Memperbarui Produk ---');
produk1.nama = 'Keyboard Mekanik';
produk1.harga = 1200.0;

print('Nama Produk (setelah update): ${produk1.nama}');
print('Harga Produk (setelah update): Rp ${produk1.harga.toStringAsFixed(2)}');

// Menguji validasi setter
print('\n--- Menguji Validasi Setter ---');

print('Mencoba mengatur nama menjadi "TV"');
produk1.nama = 'TV';
print('Nama Produk (setelah gagal update nama): ${produk1.nama}');

print('Mencoba mengatur harga menjadi -100.0');
produk1.harga = -100.0;
print('Harga Produk (setelah gagal update harga): Rp ${produk1.harga.toStringAsFixed(2)}');

print('Mencoba mengatur harga menjadi 0.0');
produk1.harga = 0.0;
print('Harga Produk (setelah gagal update harga): Rp ${produk1.harga.toStringAsFixed(2)}');

print('\n--- Status Akhir Produk ---');
print('Nama Produk: ${produk1.nama}');
print('Harga Produk: Rp ${produk1.harga.toStringAsFixed(2)}');
}

class User {

String _password;

User(this._username, this._password);

bool login(String username, String password) {
