Object Oriented Programming - Dart

1. Constructor

Constructor adalah method khusus dalam class yang digunakan untuk membuat objek. Constructor otomatis dipanggil saat objek dibuat.

2. Enkapsulasi

Enkapsulasi adalah konsep dalam OOP yang digunakan untuk menyembunyikan data agar tidak dapat diakses langsung dari luar class.

Dalam Dart, private field menggunakan tanda underscore (_).

String _username; String _password;

3. Getter dan Setter

Getter digunakan untuk mengambil data dari variabel private. Setter digunakan untuk mengubah data dengan validasi tertentu.

String get username => _username; set username(String value){ if(value.length >= 5){ _username = value; } }

4. Contoh Validasi

  • Username minimal 5 karakter
  • Password minimal 8 karakter
  • Email harus mengandung @
